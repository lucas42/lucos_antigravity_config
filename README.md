# lucOS Antigravity Config
The `.agent` directory for google antigravity to use when interacting with lucOS projects.

This repository contains configuration files, custom rules, workflows, and context specifically tailored for the Google Antigravity AI coding assistant to use while working across the various repositories within the lucOS ecosystem.

## Contents
- `rules/` — Specific instructions and behaviors the agent must follow (e.g., GitHub authentication and API interaction methods to ensure proper bot identity).
